# Blueberry-Ube Latte Scrollytelling Setup Instructions

All code for your scrollytelling Next.js 14 App Router experience has been generated in the `latte-landing` directory!

1. `latte-landing/components/UbeLatteSequence.tsx` - The Framer Motion canvas logic.
2. `latte-landing/app/page.tsx` - The main scrollytelling wrapper logic.
3. `latte-landing/app/globals.css` - Custom scrollbars and midnight styling.
4. Tailwind config naturally maps your Ube colors.

**Due to OneDrive directory locking on this system, the automated installation and image extraction processes could not complete programmatically.** 

To see your experience, please open a terminal and run the following manually:

```bash
cd latte-landing
npm install
node ../process_images.js
npm run dev
```

This will prepare the 120-frame `.webp` sequence and start your luxury frontend experience on `localhost:3000`.
Happy scrollytelling!
